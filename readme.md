# 东方札 Touhouhanafuda
一款以《东方Project》为题材、结合花札机制的原创牌类游戏。

## 🎴 简介
东方札 是一个以《东方Project》系列角色与场景为基础的卡牌游戏，灵感来源于传统日本游戏「花札」。玩家通过收集特定组合的卡牌获得分数，策略与运气兼具。目前做了简单的测试程序，直接运行 main.py 可以与只会出自己手牌第一张的BAKA对战。可以简单修改让自己左右脑互博。
